# Backgrounds

This folder holds maps, grids, space fields, station backgrounds, and reusable backdrop art.

Backgrounds should support the RTS command feeling without distracting from controls or status.

## Suggested Background Types

```text
world_map_grid
space_field
station_interior_base
room_floor_tile
tactical_overlay_grid
sensor_map_background
terminal_backdrop
```

## Rules

- Keep backgrounds low-noise.
- Controls and text must stay readable on top.
- Avoid baked labels.
- Prefer reusable tiles and overlays.
- Support dark mode by default.
- Keep file sizes reasonable.

## Naming Examples

```text
background_world_map_grid_dark_01.png
background_space_field_subtle_01.png
background_room_floor_tile_dark_01.png
background_sensor_map_grid_01.png
```
