# Concepts

This folder holds exploratory visual material for Agent Mesh.

Concepts are not final production assets. They are used to discover the look, compare directions, and generate source material for cropped sprites, rooms, stations, effects, and UI pieces.

## What Belongs Here

- Full asset sheets
- Moodboards
- Visual experiments
- World map mockups
- Station layout ideas
- Room layout ideas
- Agent character explorations
- UI/HUD explorations

## What Does Not Belong Here

- Cropped production sprites
- Final runtime UI files
- Optimized WebP exports
- App code

## Naming

Use:

```text
concept_[asset_family]_[description]_v[number].png
```

Examples:

```text
concept_ui_asset_sheet_v1.png
concept_agent_roles_v1.png
concept_station_icons_v1.png
concept_room_interiors_v1.png
concept_effects_v1.png
```

## Workflow

1. Generate broad concept sheets here.
2. Pick useful assets.
3. Crop them into the correct production folder.
4. Rename cropped assets using `ASSET_NAMING.md`.
5. Keep the original concept sheet for reference.
