# Appearance Mockup Prompt - Full UI

Use this prompt to generate a polished full-screen HUD/control surface concept.

```text
Create a high-resolution full-screen UI mockup for Agent Mesh, a lightweight RTS-style command interface for a real AI homelab.

Scene:
A polished dark sci-fi RTS control screen combining a tactical map, selected station details, active agents, alerts, and command controls.

Visual style:
- dark sci-fi RTS command HUD
- clean tactical panels
- industrial server-room influence
- cyan/blue primary glow
- purple AI accents
- yellow warning accents
- red error accents
- subtle grid background
- clipped-corner panels
- thin borders
- readable spacing
- no copyrighted logos or recognizable game UI
- no readable text baked into the image

Layout:
- top header/breadcrumb strip with blank text regions
- center tactical map or station map
- left list panel with blank rows
- right detail inspector panel with blank metric slots
- bottom active task/command bar with empty buttons
- small alert area with icon placeholders but no text

Important:
- Do not include readable words, labels, numbers, logos, or brand names.
- Leave text areas blank for app-rendered live data.
- Prioritize readability over decoration.
- Medium detail, low-noise, low-end hardware friendly.
- Make it feel like a real control interface, not just wallpaper.
```
