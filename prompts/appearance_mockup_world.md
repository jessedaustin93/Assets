# Appearance Mockup Prompt - World Map

Use this prompt to generate a visual concept for the Agent Mesh world map.

```text
Create a high-resolution UI concept mockup for Agent Mesh, a lightweight RTS-style command interface for a real AI homelab.

Scene:
A dark sci-fi tactical world map showing multiple homelab stations as command nodes. The map should feel like an RTS strategy overview, not a normal business dashboard.

Visual style:
- dark sci-fi RTS command map
- industrial/holographic interface
- clean readable panels
- cyan/blue tactical glow
- purple AI accents
- yellow warning accents
- red error accents
- subtle grid background
- station nodes with status rings
- thin connection lines between nodes
- no copyrighted logos or recognizable game UI
- no readable text baked into the image

Layout:
- center: world/network map with station nodes
- left: compact station list panel with blank rows, no text
- right: selected station detail panel with blank areas for app-rendered text
- bottom: task/command bar with empty button frames
- top: breadcrumb/header bar with no text

Stations to imply visually:
- AI flagship server station
- media/infrastructure server station
- sensor outpost
- home automation node
- kiosk node
- small recon nodes

Important:
- Do not include readable words, labels, numbers, logos, or brand names.
- Leave all text areas blank so the app can render live text.
- Make it functional and readable, not overly cluttered.
- Low-end hardware friendly, medium detail, clear silhouettes.
```
