# Backgrounds Base Prompt

Use this prompt to generate reusable Agent Mesh backgrounds.

```text
Create a clean background asset sheet for Agent Mesh, a lightweight RTS-style command interface for a real AI homelab.

Style:
- dark sci-fi tactical interface backgrounds
- subtle grid overlays
- industrial station floor patterns
- low-noise readable surfaces
- cyan/blue faint holographic accents
- no readable text, no labels, no logos
- original designs only
- low-end hardware friendly

Include background types:
1. world map dark tactical grid
2. space/network map backdrop
3. station interior floor tile
4. modular room floor tile
5. sensor/radar map background
6. terminal/log panel backdrop
7. low-power plain dark background
8. subtle holographic overlay grid
9. storage bay floor pattern
10. engineering bay floor pattern
11. AI core floor pattern
12. communications map table background

Requirements:
- backgrounds must not overpower text or controls
- keep contrast low enough for UI panels to remain readable
- no baked labels or numbers
- avoid busy details behind log/terminal areas
- separated assets for cropping
- tileable variants where useful
```
