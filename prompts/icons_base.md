# Icons Base Prompt

Use this prompt to generate Agent Mesh status, service, and control icons.

```text
Create a clean icon asset sheet for Agent Mesh, a lightweight RTS-style command interface for a real AI homelab.

Style:
- dark sci-fi tactical HUD icons
- simple readable silhouettes
- consistent stroke weight
- cyan/blue default accents
- green healthy icons
- yellow warning icons
- red error icons
- purple AI/model icons
- orange build/heat icons
- no readable text, no labels, no logos
- original designs only
- separated icons for easy cropping
- transparent background preferred
- low-end hardware friendly

Include status icons:
1. healthy
2. busy
3. warning
4. error
5. critical
6. offline
7. blocked
8. complete
9. failed
10. selected

Include system icons:
1. CPU
2. RAM
3. storage
4. GPU
5. network
6. VPN
7. SSH
8. logs
9. terminal
10. service
11. container
12. backup

Include service icons:
1. media server
2. ARR cargo/downloads
3. password vault
4. radar/signal scan
5. AI core/model
6. engineering/repair
7. communications
8. storage room
9. build lab

Include interaction icons:
1. open
2. inspect
3. assign
4. pause
5. resume
6. stop
7. restart
8. settings
9. confirm
10. cancel
11. delete/danger

Important:
- Do not use company logos.
- Do not include readable text.
- Use shape plus color so status remains understandable.
- Keep icons usable at small sizes.
```

```text
concepts/concept_icons_v1.png
```
