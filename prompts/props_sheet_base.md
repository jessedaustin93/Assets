# Props Sheet Base Prompt

Use this prompt to generate reusable room props for Agent Mesh.

```text
Create a clean game asset sheet of reusable sci-fi room props for Agent Mesh, a lightweight RTS-style command interface for a real AI homelab.

Style:
- dark sci-fi industrial station props
- readable RTS interior objects
- clean silhouettes
- cyan/blue tactical lighting
- subtle purple AI accents where appropriate
- yellow warning accent variants
- red error accent variants
- original designs only
- no readable text, no labels, no logos
- separated props for easy cropping
- transparent background preferred
- low-end hardware friendly

Perspective:
- top-down or slight isometric
- consistent perspective across all props

Include props:
1. terminal console
2. server rack
3. GPU rack
4. model storage rack
5. vault door
6. lock console
7. radar screen
8. antenna mast
9. network map table
10. media screen
11. storage drive rack
12. backup crates
13. cargo crates
14. sorting conveyor
15. repair bench
16. build/test terminal
17. drone dock
18. warning beacon
19. service control panel
20. logs terminal

States/variants where useful:
- idle
- active
- warning
- error
- locked
- offline

Important rules:
- No readable text or labels baked into the image.
- Leave screen/panel areas blank for app-rendered text.
- Keep each prop separated with padding for cropping.
- Avoid copyrighted logos or recognizable branded equipment.
```
