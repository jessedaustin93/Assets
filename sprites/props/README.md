# Prop Sprites

This folder holds reusable objects that can be placed inside rooms.

Props help rooms feel functional without requiring a whole new background for every service.

## Suggested Props

```text
terminal
server_rack
gpu_rack
vault_door
radar_screen
antenna
crate
storage_silo

drone_dock

media_screen

warning_beacon
```

```text
prop_terminal_cyber_idle_01.png

```

## Design Notes

- Props should be reusable across rooms.
- Avoid baked labels.
- Keep scale consistent.
- Prefer transparent backgrounds.
- Props can have simple states such as idle, active, warning, locked, or offline.
